02

SCANSIONE PORTE

Questo modulo serve a vedere quali servizi sono attivi su un computer target (es. un altro dispositivo nella tua rete o una macchina virtuale di laboratorio). Inserisci l'IP Target (es. 192.168.1.50). Clicca su AVVIA OPERAZIONE. La console mostrerà in tempo reale se porte come la 22 (SSH), 80 (HTTP) o 3306 (MySQL) sono aperte.

03

SCANSIONE VERBI HTTP

Utilizzalo per testare la sicurezza di un server web. Inserisci l'IP di un server web locale. Il tool invierà richieste GET, POST, PUT e DELETE. Analisi: Se vedi Esito: True su DELETE o PUT, il server potrebbe essere vulnerabile perché permette a chiunque di caricare o cancellare file.

04

SCANSIONE SOCKET

Questo trasforma il tuo PC in un ascoltatore invisibile di pacchetti. Inserisci il tuo IP locale su Kali (controllalo prima con il comando hostname -I). Clicca su AVVIA OPERAZIONE. Il log inizierà a stampare ogni pacchetto che transita sulla tua scheda di rete, indicando la provenienza e la dimensione in byte. Per fermare: Clicca di nuovo sul tasto "Avvia" o torna al menu principale.
